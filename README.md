# ROBOTIC-ARM-USING-ECG-SENSOR-
A FIRST YEAR PROJECT WITH AN OBJECTIVE TO BUILD AND PROSTHETIC ARM USING ELECTROCARDIOGRAM PULSE 


##  Overview

This project presents a **low-cost robotic arm controlled using muscle signals (EMG)**.
It is designed mainly to assist **physically challenged individuals** by converting muscle movements into robotic actions.

The system integrates **electronics, programming, and mechanical design** to create an intelligent assistive device.

---

##  Objective

* To develop an **affordable robotic arm**
* To control the arm using **muscle signals (EMG)**
* To assist **handicapped or disabled individuals**
* To combine **ECE + Mechanical + Computer Science concepts**

---

##  Working Principle

* Muscle movement generates **electrical signals**
* These signals are captured using the **AD8232 ECG/EMG sensor**
* The microcontroller processes the signal
* Based on threshold values:

  * Hand opens ➝ Robot opens
  * Hand closes ➝ Robot closes

---

##  Technologies Used

* Embedded C / Arduino Programming
* Signal Processing (Basic EMG)
* Robotics

---

##  System Architecture

* Input: Muscle Signal (EMG)
* Processing: ATmega328
* Output: Servo Motors controlling robotic fingers

---

##  Code Explanation

The system works using:

* Threshold-based signal detection
* Servo motor control for each finger

Example logic:

* If signal > threshold → Close hand
* Else → Open hand
